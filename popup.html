About dencoder

dencoder is used to compute cryptographic hashes, inspect payloads, and run common developer conversions. It is entirely implemented in JavaScript and all calculations are performed on the client side.

  • Hash: MD5, SHA-1, SHA-2 (224, 256, 384, 512), RIPEMD-160, MD4, Whirpool
  • HMAC: MD5, SHA-1, SHA-2 (224, 256, 384, 512), RIPEMD-160, MD4
  • CRC: CRC-8, CRC-16, FCS-16, FCS/CRC-32
  • Cipher (interoperable with OpenSSL): AES-256, DES, Triple DES, Rabbit, RC4, RC4Drop. CBC/Pkcs7 is used.
  • Net: Subnet calculator, Ip ↔ Dec, Ip → Bin, Ip → Hex
  • Time: Unix ↔ Datetime, Unix ↔ RFC-1123, Unix ↔ ISO 8601
  • Numbers: Dec ↔ Hex, Dec ↔ Bin, Dec ↔ Roman
  • Strings: ASCII ↔ Hex, UTF-8 ↔ Hex, UTF-16 ↔ Hex
  • Encode: Base64, Base32, Base45, Base58, ROT-13, JavaScript encodeURI(), encodeURIComponent(), HTML special chars encode/decode, 텍스트 ↔ 유니코드 변환
  • JWT / JSON / Morse: JWT encode/decode and header mutation payloads, JSON beautifier, dict beautifier, Morse encode/decode
  • Password: Password strength scoring and entropy analysis

Usage: enter text and click on the result. It will be copied to clipboard.

Extension is using crypto-js and other open-source libraries.
The current dencoder build is based on the legacy Hasher workflow and keeps all processing local.

Source: github.com/sm1ee/dencoder
