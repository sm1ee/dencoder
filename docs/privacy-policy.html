dencoder Privacy Policy

Last updated: 2026-03-09

dencoder processes developer input locally inside the extension popup or extension tab.

Data Collection

dencoder does not collect, transmit, sell, or share personal data with the developer or third parties.

Data Processing

  • Text entered into the extension is processed locally in the browser.
  • Generated output can be copied to the clipboard when the user chooses to do so.
  • dencoder does not use remote analytics, advertising, tracking pixels, or external APIs for feature execution.

Permissions

  • clipboardWrite: used only when the user copies generated output from the extension.

Storage

dencoder does not intentionally store submitted text on remote servers.

Third-Party Libraries

dencoder bundles third-party libraries inside the extension package for local execution. It does not execute remotely hosted code.
